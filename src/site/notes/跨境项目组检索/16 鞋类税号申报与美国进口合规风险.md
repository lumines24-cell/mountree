# 鞋类税号申报与美国进口合规风险

## 资料属性

来源文本标注为“AI 生成，不能完全保障真实”。本笔记只作内部风险提示。税号、HTS、关税、FCA、进口政策属于高风险合规事项，正式申报前必须由报关行、美国 customs broker 或贸易律师确认。

## 中国出口税号参考

### 核心分界

- 真皮鞋：鞋面外表面积 50% 以上为天然动物皮革，通常归入 6403
- 人造革 / 合成革鞋：PU、PVC、超纤等塑料 / 高分子鞋面，通常归入 6402

## 申报要素

报关资料必须写明：

- 鞋面材质：真皮 / PU / 超纤 / 纺织 / 其他
- 外底材质：橡胶 / TPR / EVA / 其他
- 是否过踝
- 是否运动鞋
- 鞋面外表面积占比
- 原产国
- FOB 货值
- 数量、箱数、型号

## 美国进口税费参考

资料中提到的参考口径：

- 美国进口清关通常无联邦 VAT
- 进口计税基数通常按 FOB 货值
- 小额免税 Section 321 规则存在，但适用范围和政策需实时确认
- MPF、HMF 等行政费可能适用

资料中提到的美国进口税率、122 条款、301 税率等信息具有高度时效性。正式核算必须以美国 HTS、CBP、报关行当期口径为准。

## FCA / 虚假申报风险

美国 False Claims Act / FCA 可能覆盖对美国联邦政府的虚假申报和反向虚假申报。

高风险行为：

- 低报货值
- 拆分货值
- 伪造发票
- 真皮鞋与人造革鞋混淆申报
- 虚报原产国
- 伪造原产地证
- 瞒报数量
- 虚报样品 / 低值货
- 虚假申报 Vegan、GRS、环保认证骗取补贴或平台权益

风险特点：

- 不一定需要主观故意，草率漠视真实性也可能构成风险
- 内部员工、货代、美仓、合作方都可能成为举报来源
- 一旦涉及美国海关和联邦资金，风险成本远高于普通行政罚款

## Mountree 合规要求

必须建立：

- 材质检测报告
- 采购发票
- 供应商声明
- 鞋面材质比例记录
- FOB 货值计算底稿
- 报关单证复核流程
- 原产地证明
- 物流、报关、付款记录

留存建议：

- 至少 5 年

## 禁止操作

- 不得把 PU / 超纤鞋申报为真皮鞋
- 不得为了低税率更改材质描述
- 不得低报 FOB
- 不得把商业货物拆成样品规避税费
- 不得无证据宣称整鞋 Vegan、GRS 或环保认证
