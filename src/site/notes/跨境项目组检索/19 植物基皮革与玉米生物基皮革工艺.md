## 玉米生物基皮革两条路线

### 路线 1：生物基 PU 玉米皮革

适合 Mountree 优先关注。

工艺链条：

1. 玉米淀粉提取
2. 酶解糖化为葡萄糖
3. 微生物发酵制备生物 PDO
4. 生物 PDO + MDI + 多元醇聚合成生物基 PU 树脂
5. 离型纸涂布成革
6. 压纹、揉软、防水防污后整理
7. 耐折、耐磨、剥离强度、水解稳定性检测

优点：

- 工业量产稳定
- 耐弯折较好
- 手感接近中厚牛皮
- 制鞋厂设备改造小
- 适合乐福、牛津、德比、切尔西靴、通勤女鞋

关注指标：

- 生物基碳含量
- 耐折次数
- 耐磨
- 剥离强度
- 水解稳定性
- REACH、DMF、重金属检测

### 路线 2：PLA 全淀粉改性皮革

不建议作为 Mountree 主力鞋面。

优点：

- 植物基含量高
- 堆肥环境下可降解潜力
- 成本较低

短板：

- 低温易脆
- 潮湿环境强度下降
- 久穿易裂
- 手感塑料感更明显

适合：

- 低频休闲鞋
- 配饰
- 非高强度弯折部件

## PDO 与 PLA 区分

PDO = 1,3-Propanediol / 1,3-丙二醇。

- 小分子醇
- 用于合成生物基 PU
- 决定柔韧、回弹和耐折性能

PLA = Polylactic Acid / 聚乳酸。

- 本身是高分子材料
- 可熔融压膜
- 更硬、更脆，耐水性和耐折较弱

简化判断：

- 玉米淀粉 → 发酵 PDO → 生物基 PU：更适合鞋面
- 玉米淀粉 → 发酵乳酸 → PLA：更适合低频、低强度应用

## Mountree 选材建议

优先级：

1. 玉米生物基 PU：量产稳定，适合 $79-$119 价格基准
2. 仙人掌 / 苹果皮革：用于故事款、拍摄款、轻溢价款
3. 菠萝革：用于休闲款和测试款
4. PLA / Mylo：暂不作为主力量产

采购必须要求：

- 生物基含量检测
- 耐折测试
- 耐磨测试
- 水解测试
- 色牢度
- 剥离强度
- 鞋面开料和成型试产反馈
