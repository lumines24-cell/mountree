# Impact 与 LTK 联盟红人平台对比

## 资料属性

来源文本标注为“AI 生成，不能完全保障真实”。本笔记用于渠道工具选型参考，平台功能、费用和集成能力需以上线时官方信息为准。

## 核心区别

| 维度 | impact.com | LTK |
|---|---|---|
| 类型 | 开放式合作伙伴营销 SaaS | 封闭式时尚生活红人购物生态 |
| 自有 C 端流量 | 无 | 有 LTK App 购物用户 |
| 合作对象 | 红人、联盟、返利、老客推荐、B2B | LTK 平台内时尚生活博主 |
| 控制权 | 品牌自主管合约、佣金、归因、数据 | 平台托管为主 |
| 适合 | 全域联盟和长期红人池 | 女鞋穿搭种草和即时带货 |
| 短板 | 没有自带购物流量 | 生态封闭，外部博主兼容弱 |

## impact.com 适合 Mountree 的场景

- Shopify / WooCommerce 独立站联盟
- Amazon 站外联盟归因
- 外部 TikTok / Instagram / YouTube 博主分佣
- 返利网站、优惠券站、测评博客
- 老客推荐和会员裂变
- 多国家、多币种、多产品线统一结算

适合阶段：

- 有基础独立站后
- 想自建长期红人池
- 想管理佣金、退款、复购、新客奖励
- 需要把红人、Affiliate、分销统一到一个后台

## LTK 适合 Mountree 的场景

- 北美女鞋穿搭博主种草
- 通勤鞋、乐福鞋穿搭内容
- 借助 LTK App 购物流量
- 中腰部穿搭博主带货
- 希望减少自主管理成本

适合阶段：

- 已有较完整产品图、尺码、库存
- 需要快速切入时尚穿搭内容池
- 有预算做中腰部红人合作

## 推荐组合

多数女鞋品牌可采用：

- LTK：抓高转化穿搭博主和平台购物流量
- impact.com：承接全域大盘，自建红人、返利、老客推荐和 B2B 分销

冷启动阶段建议：

1. 先用手动红人表 + Shopify 折扣码测试
2. 达到稳定订单后接入 impact.com 或轻量工具
3. 有稳定素材和库存后测试 LTK / ShopMy
