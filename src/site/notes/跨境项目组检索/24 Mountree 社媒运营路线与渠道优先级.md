# Mountree 社媒运营路线与渠道优先级

## 文件定位

本笔记整理自 `社媒运营路线规划.png` 和 PPT 中的社媒平台矩阵，用于明确不同平台在 Mountree 起盘中的角色。

## 渠道总逻辑

- TikTok：爆发与新品测试。
- Pinterest：长期搜索流量和购买意图拦截。
- Instagram：品牌信任和高级视觉。
- Facebook：再营销、DPA、购物车挽回。
- YouTube / Shorts：长效 SEO、深度测评和信任补充。
- X：观点种草、SEO 和 KOL 扩散。

## 推荐优先级

### 第一优先级：TikTok

角色：

- 病毒传播
- 内容测试
- 达人种草
- 爆款角度验证

链路：

- 爆款视频
- Spark Ads 放大
- Affiliate / 达人挂链
- TikTok Shop 或独立站转化
- 外溢到 Amazon 品牌词搜索

适合内容：

- 弹力拉伸测试
- 气垫缓震测试
- 久站痛点对比
- 通勤场景短剧
- GRWM / OOTD
- 真实试穿反馈

### 第二优先级：Pinterest

角色：

- 女鞋强种草
- 搜索 / 推荐流量
- 长期内容资产
- 中高购买意图流量

链路：

- Pin 内容
- Pinterest 搜索 / 推荐
- Landing Page
- 订阅 / 加购
- 成交

落地页排序：

1. Shopify 独立站
2. 品牌官网
3. Amazon Brand Store
4. TikTok Shop

### 第三优先级：Instagram

角色：

- 品牌形象
- Reels 提量
- Shop 标签
- 高级视觉与信任建立

流量优先级：

1. Reels
2. Explore
3. Feed
4. Stories

内容类型：

- Try-on
- Outfit Styling
- Aesthetic Brand Content
- Influencer 合作

### 第四优先级：Facebook

角色：

- 精准广告
- 再营销
- 社群运营
- 转化收割

适用前提：

- 已有 Landing Page
- 有流量基础
- 有加购数据

重点广告：

- DPA 动态产品广告
- 购物车再营销
- Lookalike 相似人群扩量

### 第五优先级：YouTube / Shorts

角色：

- 长效 SEO
- 深度种草
- 信任补充

内容类型：

- Try-on
- Outfit transformation
- Review
- 开箱视频
- 讲理由的长内容

## 第二梯队平台

- Whatnot：直播平台，可观察但不作为当前核心。
- LTK：适合穿搭红人和高质量内容种草。
- Lemon8：小红书海外版，可作为审美内容测试场。

## 执行原则

- 当前价格基准仍为 `$79-$119`。
- TikTok 负责爆发，Pinterest 负责长期，Instagram 负责信任，Facebook 负责收割。
- Amazon 不作为社媒内容的第一落点，优先承接品牌词和外溢搜索。
