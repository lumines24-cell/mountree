# Mountree 品牌调性基准

> 本页是 Mountree 视觉、文案、内容与 AI 生成任务的固定调性基准。先使用本页，再叠加具体项目需求。

## 一句话锚点

Mountree creates plant-based comfort footwear for women in motion.

**Walk Strong. Bend Gracefully.**

任何设计或文案决策，先问：这个选择是让品牌更安静高级，还是更喧哗廉价？

### 最小尺寸与安全边距

| 版本 | 最小使用尺寸 | 安全边距 |
|---|---|---|
| 徽章版 | 48px（数字端）/ 15mm（印刷端） | 四周留白 ≥ Logo 直径的 25% |
| 横版 Wordmark | 高度 24px（数字端）/ 8mm（印刷端） | 左右留白 ≥ 字母 “M” 宽度的 1 倍 |

低于最小尺寸时，优先使用徽章版；其结构更简单，小尺寸下识别度更高。

### 使用规则

**Do**

- 直接使用官方导出的透明底 PNG / SVG，不重新描摹。
- 放置在纯色或低对比度渐变背景上；深松绿、米白、暖沙均可。
- 始终等比缩放。

**Don't**

- 不拉伸、压扁或旋转。
- 不添加阴影、描边、发光或渐变叠加。
- 不更换正式色彩系统之外的颜色。
- 不压在高对比图案、照片人脸或高饱和色块上。
- 不将徽章版拉长作横幅，也不将横版压缩作头像；两个版本不可互相变形替代。

## 品牌气质

- 沉静（Calm）
- 克制（Restrained）
- 真实（Authentic）
- 有质感（Tactile）
- 不喧哗（Understated）

自然、优雅、专业、轻奢；舒适但不运动化，女性化但不甜腻，植物基材料但不说教。

## 包装视觉语言

本规范依据已审阅的鞋盒与内包装纸初稿整理。以下仅固定与 Mountree 调性一致的视觉元素；打样、材质、成本与运输结构仍需单独确认。

### 固定采用

| 元素 | 正式方向 | 使用边界 |
|---|---|---|
| 盒体主配色 | 深松绿外盒搭配米白内衬；或米白盒盖搭配深松绿底盒 | 哑光、低饱和、自然纸感；不使用高光覆膜或高对比拼色 |
| Logo 落位 | 横版 Wordmark 单独居中，保留大面积留白 | 一个可视面只使用一个主 Logo；不与大段文案竞争 |
| 鞋履线稿 | 鞋形与枝叶相连的线稿，作为角落压纹或低对比内页纹样 | 线稿只作辅助，不超过单一面的约 15% 视觉面积 |
| 表面工艺 | 同色压凹 / 压凸，或低对比 UV 局部工艺 | 以触感和光线变化表达层次，不以闪亮效果制造廉价“奢华感” |
| 内包装纸 | 米白底配浅苔或暖沙的低对比鞋履线稿重复纹样 | 优先“图形重复”，避免大面积、密集的 Wordmark 重复 |
| 封口方式 | 内包装纸折口使用圆形徽章贴纸 | 贴纸采用深松绿底配米白标识，或米白底配深松绿标识；保持单色、无金属高光 |
| 开箱层次 | 外盒、米白内包装纸、圆形封口贴、护理卡依次展开 | 层次服务于安静的仪式感，不增加无功能的装饰件 |

### 可选的高端 SKU 元素

- 米白绒布 / 布感收纳袋，搭配深松绿抽绳与低对比 Wordmark。
- 护理卡使用米白底、深松绿正文与小面积鞋履线稿；内容控制为简短的产品护理信息。

### 不纳入固定基准

- 金色烫印不作为默认工艺：它会引入正式色板之外的高反光金属感；如未来用于限量款，须先单独确认。
- 密集、斜向、满版的 Wordmark 重复纹样：容易削弱“自然留白”和安静感。
- 山、树、叶等多种辅助图形同时出现：保留鞋履枝叶线稿即可，避免视觉叙事堆叠。
- `Where she stands, a mountain grows, a tree breathes` 不作为当前包装固定文案；包装与视频仅使用正式 Slogan：`Walk Strong. Bend Gracefully.`

### 包装打样自检

- [ ] 盒体、内衬与纸张是否只使用正式色彩系统？
- [ ] Logo 是否清晰、居中且具备足够留白？
- [ ] 图形纹样是否低对比、低密度，且没有压过 Logo？
- [ ] 表面工艺是否偏哑光和触感，而非镜面、闪亮或高反光？
- [ ] 打样实物在自然光与货架光下，是否仍然呈现克制、专业与轻奢感？

## 视觉语言

- 构图：自然留白，主体不必居中，保留呼吸感。
- 场景：真实生活场景与自然光优先，避免纯白影棚式电商感。
- 人像：自然状态，避免过强摆拍与网红姿势。
- 视频：可快剪，但转场、字幕与音效保持克制。
- 字体：英文标题使用优雅衬线字体（如 Cormorant Garamond / Georgia）；正文使用轻量无衬线（如 Jost / Avenir）。

## 视频 / 动态规范

### 字幕规范

| 项目 | 规范 |
|---|---|
| 字体 | 正文使用 Jost / Avenir；标题可用 Cormorant Garamond |
| 主字色 | 米白 `#F6F1E9`（深色背景）/ 近黑松绿 `#18201A`（浅色背景） |
| 背景衬底 | 半透明深松绿 `#3A5840` 色块，透明度 60–75%；禁止纯黑色块 |
| 最小对比度 | 字色与背景对比度 ≥ 4.5:1；米白压苔色 `#748561` 时须加深底色 |
| 字号 | 9:16 竖屏中，字高不低于画面高度的 4.5% |
| 位置 | 居中偏下，位于画面高度 70–80% 区域；避免遮挡产品主体 |
| 动效 | 淡入淡出或轻微上浮；禁止弹跳、闪烁、逐字爆破 |

### 转场规范

**可用**：硬切（straight cut）、淡入淡出（cross dissolve）、轻微推拉（slow push / pull，速度 ≤ 1.05x / 秒）。

**禁用**：甩动模糊（whip pan）、缩放炸裂、故障风（glitch）、闪白、每秒超过 3 次的连续快速切换，以及网红感贴纸或表情包转场。

### 音效与配乐

- 背景音乐优先低频、氛围感、无强烈鼓点的 lo-fi 或 acoustic。
- 禁止卡通音效（如叮咚、爆炸）与过度戏剧化的 bass drop。
- 人声 / 口播保持平稳语速，避免带货腔的上扬语调。

### 品牌签名动作（Intro / Outro）

- **开场**：徽章版 Logo 淡入，停留不超过 1 秒；不使用旋转或弹入动画。
- **结尾**：横版 Wordmark 淡入，配合 `Walk Strong. Bend Gracefully.`；米白字置于深松绿底。
- 所有视频统一采用这套 Intro / Outro，保持 TikTok、Instagram、Pinterest 的跨平台识别一致性。

## 文案语言

语气使用短句、平静、具体的表达，先说舒适、灵活与日常适配，再谈更有意识的材料选择。

可参考：

- 一双鞋，走得久一点。
- 苔色，是雨后森林的颜色。
- 简单的选择，留给复杂的日子。

避免：

- 网红夸张腔：如“OMG”“姐妹们”“绝对想不到”。
- 环保说教腔：如“拯救地球，从脚下开始”。
- 硬广叫卖腔：如“限时五折，先到先得”。
- 身材焦虑表达：如 `slimming legs`、`skinny legs`、`hides thick calves`。

可使用：`sleek silhouette`、`tailored fit`、`gap-free fit`、`second-skin stretch`。
