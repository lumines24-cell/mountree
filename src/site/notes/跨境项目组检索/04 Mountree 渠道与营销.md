# Mountree 渠道与营销

## 渠道结构

Mountree 当前适合采用三渠道并行：

| 渠道 | 角色 | 重点 |
|---|---|---|
| TikTok Shop | 起盘引擎 | 达人种草、短视频挂车、爆款内容测试 |
| Amazon | 信任承接 | 承接搜索、评价积累、品牌词防御 |
| Shopify 独立站 | 品牌阵地 | 利润沉淀、品牌故事、会员复购 |

当前渠道总方向：项目以 $79-$119 为价格基准，采用本土化履约体系，避免低端跨境直发模式。

## TikTok Shop

定位：

- 起盘引擎
- 内容测款平台
- 站外种草核心阵地

已记录策略：

- 达人种草 + 短视频挂车
- 如使用高于当前成交基准的锚价，需单独确认并记录为测试策略
- 实际成交以 $79-$119 为当前基准
- 达人分批合作

内容角度：

- 奢牌平替
- 久站测评
- 利落线条与通勤造型
- 通勤一整天不累
- 护士/教师/职场女性场景

## Amazon

定位：

- 信任承接
- 品牌词防御
- 承接 TikTok 外溢搜索

已记录策略：

- 深度整合 Amazon 生态
- 独立站开通 Buy with Prime，提高转化率
- 使用 Amazon MCF / Multi-Channel Fulfillment 实现北美本土高效履约
- 数据好的款再入美仓
- TikTok 种草后引导用户搜索品牌词或产品词

## Shopify 独立站

定位：

- 品牌阵地
- 利润蓄水池
- 完整品牌故事表达
- 会员复购沉淀

已记录：

- 利润率最高，约 36-44%
- 使用 Shopify Dawn 模版方向
- 首页 + About 页优先
- 风格参考“山下有松”的克制美学
- 第一阶段使用 Shopify 快速起量、视觉呈现和品牌测款
- 中长期规划迁移到 WordPress / WooCommerce，以获得更高自主权、深度定制能力和更低长期抽成成本

TikTok → IG Reels → Pinterest

## 运营路线


- 垂直精铺矩阵 + 站外种草品牌结合
- 先在社媒测款
- 数据好再入 Amazon
- 前期 TikTok + 独立站种草验款
- 数据好的款入美仓
- Amazon + TikTok 重点投放

- 选款力
