# Mountree 视觉与文案系统

## Logo 记忆

Mountree Logo 特征：

- 深绿色
- 鞋形结合枝叶元素
- 品牌名为 Mountree
- 视觉联想：mountain + tree，自然、成长、优雅、女性鞋履

## 主色方向

当前偏好：

- 松绿
- 苔色
- 米白
- 低饱和自然色

已记录松绿版颜色方向：

| 用途 | 色彩 |
|---|---|
| 公告栏 / 徽章 | 深松绿 `#3A5840` |
| Hero 图块 | 苔绿渐变，浅苔 → 中松 → 深松 |
| Footer | 近黑松绿 `#18201A` |

旧版大地色方向：

- 米白 `#F6F1E9`
- 暖沙 `#D4C9B0`
- 大地棕 `#5C4E3D`
- 近黑 `#1C1714`

当前建议以松绿版为主。

## 视觉关键词

- 克制美学
- 自然留白
- 低饱和
- 文学感
- 轻奢
- 现代女性
- 慢生活
- 不夸张、不廉价、不堆砌

## 字体方向

已出现方向：

- 展示标题：衬线字体，类似 Cormorant Garamond / Georgia
- 正文：轻量无衬线，类似 Jost / Avenir

建议：

- 英文标题使用优雅衬线
- 正文使用干净无衬线
- 避免过度花体，避免太商业海报感

## 品牌 Slogan

正式记忆：

> Walk Strong. Bend Gracefully.
