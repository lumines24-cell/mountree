# 植物基鞋底材质与采购验收参考

## 资料属性

来源文本标注为“AI 生成，不能完全保障真实”。本笔记用于材料认知、供应商沟通和采购验收框架。正式采购需以第三方测试报告和供应商实测数据为准。

## 植物基鞋底定义

植物基鞋底指原料主要取自甘蔗、橡胶树、蓖麻、稻壳、木薯等可再生植物，用于替代部分石油基 EVA、合成橡胶、PU 的鞋底体系。

主流类别：

- 天然植物橡胶
- 生物基发泡塑料
- 植物纤维复合材料
- 全植物混炼胶

## 主流材料

### 天然橡胶

来源：橡胶树乳胶。

适合：

- 大底
- 防滑耐磨区域
- 休闲鞋、靴子、乐福鞋外底

优点：

- 弹性强
- 防滑耐磨
- 抓地力好
- 植物来源

短板：

- 重量高于 EVA
- 低温易变硬
- 单独缓震较弱，常搭配发泡中底

### 生物基 EVA / 甘蔗 EVA

来源：甘蔗发酵乙烯制备。

适合：

- 中底
- 缓震层
- 通勤鞋、休闲鞋、运动休闲款

优点：

- 轻质
- 发泡工艺成熟
- 弯折回弹好
- 可用传统 EVA 设备生产

注意：

- 必须确认生物基含量
- 不要笼统宣传“天然”或“可降解”

### 蓖麻油基 PA / Pebax

来源：蓖麻籽油脂聚合。

适合：

- 支撑件
- 后跟稳定片
- 弹性支撑块
- 高性能轻量部件

优点：

- 耐低温
- 高韧性
- 抗疲劳形变

短板：

- 成本较高
- 对 Mountree 当前 $79-$119 价格基准需谨慎评估

### PLA 聚乳酸

来源：玉米 / 木薯淀粉发酵乳酸聚合。

适合：

- 硬质复合底
- 凉鞋底
- 低频轻量平板底

优点：

- 植物来源
- 堆肥环境下有降解潜力
- 硬度高、定型好

短板：

- 潮湿易水解变脆
- 低温韧性差
- 缓震弱
- 不建议作为 Mountree 主力通勤鞋中底

### 农业废料复合填充

材料：

- 稻壳粉
- 椰子油
- 松木树脂
- 麦麸
- 竹纤维

作用：

- 降低树脂用量
- 提升天然质感
- 消化农业边角料
- 可作为卖点辅助，但不能夸大为全植物或可降解

### 小众前沿材料

- Cork 软木：适合鞋垫或薄休闲底，不适合耐磨大底
- PHA：可降解潜力强，但成本高
- 菌丝体复合：仍偏小批量测试

## 植物基 vs 石化鞋底

| 维度 | 植物基鞋底 | 传统 EVA / 合成橡胶 |
|---|---|---|
| 原料 | 甘蔗、橡胶树、蓖麻、淀粉 | 石油、煤炭 |
| 碳足迹 | 潜在更低，需检测证明 | 通常较高 |
| 降解性 | 视材料和环境而定 | 多数难自然降解 |
| 重量 | 生物 EVA 接近传统 EVA，天然橡胶更重 | 成熟稳定 |
| 耐磨 | 天然橡胶较好，PLA 较弱 | 合成橡胶成熟 |
| 耐水解 | PLA 弱，EVA/橡胶稳定 | 整体成熟 |
| 成本 | 通常高 15%-40% | 成本低、供应成熟 |

## Mountree 采购建议

当前优先级：

1. 生物基 EVA 中底：用于轻量缓震
2. 天然橡胶 / 稻壳复合外底：用于防滑耐磨和材料故事
3. 蓖麻 PA 支撑件：用于后跟、足弓或稳定片
4. PLA：谨慎，仅用于非高弯折低频部件

## 验收指标

采购必须索取：

- ASTM D6866 生物基含量报告
- 耐磨测试
- 防滑测试
- 压缩永久变形
- 回弹率
- 水解测试
- 低温弯折
- 剥离强度
- 老化测试
- REACH / CPSIA / 重金属检测，视市场和品类需要

## 合规表达

推荐：

- Bio-based EVA midsole, made with sugarcane-derived materials
- Natural rubber outsole blended with agricultural by-products
- Contains bio-based materials, verified by ASTM D6866 testing

避免：

- 100% plant-based
- Fully biodegradable sole
- Zero carbon sole
- Eco-friendly outsole
