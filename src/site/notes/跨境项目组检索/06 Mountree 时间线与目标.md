# Mountree 时间线与目标

## 第一阶段：2026 下半年 · 北美冷启动

目标：验证产品-市场匹配，建立首批内容素材、评价和销售数据。

| 时间      | 动作                            |
| ------- | ----------------------------- |
| 7-8 月   | 市场调研完成，首批 SKU 定型，备货生产  社媒内容先行 |
| 8-9 月   | 货物出口至北美海外仓，三平台同步上架            |
| 9-10 月  | 达人分批种草启动，锁定爆款内容角度             |
| 11-12 月 | Q4 Autumn/Winter 秋冬高档靴款战役，黑五旺季冲刺，验证 PMF，积累首批评价 |

## 第二阶段：2027 年 · 北美扩张

目标：放大已验证产品、内容与渠道组合。

关键动作：

- 三平台规模化运营
- TikTok / Meta / Pinterest 精准投放
- 推出植物基材料 Green Series 第二代产品线
- 启动欧洲市场前期调研

待补充：

- 月销目标
- 广告预算
- SKU 数量
- 复购目标

## 第三阶段：2027-2028 年 · 欧洲市场进入

目标：复用北美品牌资产进入欧洲。

首站：

- 英国
- 德国
- 法国

关键动作：

- 复用北美已验证的品牌资产与内容素材库
- 推出仙人掌纤维 Summit Series 旗舰线
- 申请欧盟 PETA / V-Label 可持续认证

待补充：

- 年出口额目标
- 欧洲首批 SKU
- 认证预算
- 本地仓储方案
