# Mountree 社媒 SOP 与五层红人筛选漏斗

## 文件定位

本笔记整理自 `mountree 品牌社媒SOP.png`，用于沉淀 Mountree 社媒运营的品牌核心、红人筛选漏斗、平台内容方向和风险控制。

## 品牌核心

目标受众：

- 北美高管及职场精英女性

核心卖点：

- 极致舒适
- 双层减震气垫
- U 型乳胶枕
- 护趾棉

环保价值：

- 仙人掌 / 玉米生物基环保皮革

## 五层红人筛选漏斗

### 第一层：受众物理属性过滤

重点看：

- 地域匹配
- 年龄层匹配
- 体重 / 身材 / 生活方式匹配
- 是否覆盖真实通勤、久站、职场或高质量生活方式人群

### 第二层：品牌溢价与审美匹配

重点看：

- 是否能呈现 Quiet Luxury / 职场质感 / 高级通勤
- 是否有品牌感，而不是低价快时尚气质
- 视觉内容是否干净、克制、可信

### 第三层：数据完整性与粉丝质量

重点看：

- 粉丝地域和年龄结构
- 互动率是否正常
- 评论区是否真实
- 是否有过鞋履、通勤、职场、环保、生活方式相关内容

### 第四层：专业展示能力与痛点共鸣

重点看：

- 是否能讲清楚久站痛点
- 是否能演示气垫、弹力、贴合、防滑、耐脏等功能
- 是否能自然嵌入办公、通勤、商务晚宴、城市步行等场景

### 第五层：履约风险与带货闭环

重点看：

- 是否配合寄样排期
- 是否能按 brief 交付内容
- 是否接受专属码 / 挂链 / affiliate
- 是否有内容授权和二次投放配合度
- 是否存在言论、版权、合规或虚假宣传风险
