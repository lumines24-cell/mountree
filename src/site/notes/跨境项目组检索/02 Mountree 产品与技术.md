# Mountree 产品与技术

## 产品定位

Mountree 产品应围绕“极致舒适 + 专业形象 + 高端职场通勤 + 绿色材料”构建。当前核心品类是高端职场通勤舒适高跟鞋，以及 Q4 秋冬高档长靴。

## 产品线规划

### Mountain 山 · 靴子线

定位：秋冬、通勤、城市、显瘦、久站舒适。

可能品类：

- 弹力膝高靴
- 骑士靴
- 切尔西靴
- 短靴

价格带：

- $79-$119

### Tree 树 · 跟鞋线

定位：办公室、通勤、轻正式、日常优雅。

可能品类：

- 尖头跟鞋
- 玛丽珍
- 乐福鞋
- 凉鞋

建议价格带：

- $79-$119

## 核心技术命名

### MounThread™

弹力超纤面料。

记忆点：

- 双向弹力贴合
- 适合弹力靴
- 当前阶段可作为量产稳定材料

### 双层减震气垫技术 / MountStrata™

专为高跟鞋设计的双层减震气垫技术，是 Mountree 当前核心科技壁垒。

记忆点：

- 强调久站舒适
- 可用于“12小时久站设计”表达
- 适合医疗、教育、服务业、职场女性场景
- 正在推进美国专利 US Patent 申请流程
- 用于支撑品牌差异化和 $79-$119 价格基准

### U-shaped latex pillows / U 型乳胶枕

记忆点：

- 提供精准足弓支撑
- 提供足跟支撑
- 优化高跟鞋久穿体验

### MountCradle™

Protective toe cotton / 前掌防撞护趾棉 / 前掌保护结构。

记忆点：

- 适合跟鞋和尖头鞋
- 解决前掌挤压、磨脚、长时间穿着痛点

## 环保材料路线

植物基超纤 / 玉米基 / 仙人掌基等。

重点材料储备：

- Desserto 仙人掌皮：用于提升产品故事线与高端溢价能力
- Viridis 玉米基皮革：兼顾高耐磨性、柔韧度与可持续性
- 植物基皮革、玉米生物基皮革工艺参考：[[跨境项目组检索/19 植物基皮革与玉米生物基皮革工艺\|19 植物基皮革与玉米生物基皮革工艺]]
- 植物基鞋底材料、采购验收与 ASTM D6866 生物基含量检测参考：[[跨境项目组检索/20 植物基鞋底材质与采购验收参考\|20 植物基鞋底材质与采购验收参考]]

## 注意

“环保”不应单独作为第一购买理由。北美消费者会认可环保，但最终转化更依赖舒适、款式、价格、评价和场景适配。
环保、纯素、植物基等表达必须按 [[跨境项目组检索/14 FTC 环保宣传合规\|14 FTC 环保宣传合规]] 做证据和话术校验。
