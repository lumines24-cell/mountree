# 北美纯素消费市场与 Mountree 入局机会

## 资料属性

来源文本标注为“AI 生成，不能完全保障真实”。本笔记用于市场理解和策略参考，具体规模、占比、增速需在正式使用前核验。

## 市场基本盘

北美是全球成熟纯素消费市场的核心区域。严格纯素者规模不大，但弹性素食者和价值观消费人群构成主要增长来源。

核心判断：

- 严格纯素不是唯一客群
- Flexitarian / 弹性素食者贡献更大消费增量
- 高收入城市中产更愿为纯素、无动物实验、可持续支付溢价
- 年轻世代把可持续、伦理消费作为身份标签

## 消费动机排序

1. 健康管理：减脂、控糖、控脂、慢病预防
2. 动物福利：无动物成分、无动物实验
3. 气候环保：低碳、减少畜牧业影响
4. 过敏与不耐受：乳糖不耐、食物过敏
5. 潮流与社交标签：Clean Beauty、Plant-Based Fashion、Vegan Lifestyle

## 与 Mountree 相关的鞋履机会

纯素鞋履的关键不是“纯素”单点，而是：

- 舒适
- 材质可信
- 设计好看
- 价格合理
- 认证可证明
- 不像廉价塑料

Mountree 的切入逻辑：

- 用 Vegan / 无动物成分建立价值观基础
- 用气垫、足弓、宽楦、窄筒等功能建立购买理由
- 用 $79-$119 价格基准降低尝试门槛
- 用合规材料表达避免漂绿风险

## 认证与合规

消费者认可度较高的方向：

- Certified Vegan
- PETA-Approved Vegan
- Leaping Bunny
- GRS

但必须注意：

- Vegan 不等于低碳
- Vegan 不等于可降解
- Recycled 不等于可回收
- Plant-based 不等于 100% 天然
- 所有环保表达必须标注范围、比例和证据
